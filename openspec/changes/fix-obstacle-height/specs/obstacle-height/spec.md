# Obstacle Height Adjustment

## Requirements

### R1: 大型树木高度提升
- 大树高度从 3.5-5 单位提升到 12-18 单位
- 高度分配差异化，避免所有树一样高

### R2: 中型树木高度提升
- 中树高度从 2.5-3 单位提升到 7-10 单位

### R3: 岩石高度提升
- 岩石高度从 0.6-1 单位提升到 1.5-3 单位

### R4: 灌木高度适度提升
- 灌木高度从 0.8-1 单位提升到 1.5-2.5 单位

### R5: 树桩高度适度提升
- 树桩高度从 0.8 单位提升到 1.2-1.5 单位

### R6: 树干比例调整
- 树木 trunkHeight 比例从 0.4 提高到 0.5wg

### R7: 树冠半径调整
- 树冠锥体半径倍数从 [1.2, 0.9, 0.6] 调整为 [1.0, 0.75, 0.5]

### R8: 花朵高度不变

### R9: 碰撞检测适配
- 碰撞检测通过 Box3.setFromObject 自动适配新尺寸
- 伤害检测通过 mesh.position 距离计算，不受高度影响

## Scenarios

### Scenario: 大树看起来高大真实
- **GIVEN** 玩家在游戏场景中
- **WHEN** 玩家靠近一棵大树
- **THEN** 树的高度明显超过玩家身高（8-11 倍），视觉上像真实的大树

### Scenario: 岩石不低于腰部
- **GIVEN** 玩家在游戏场景中
- **WHEN** 玩家靠近一块岩石
- **THEN** 岩石高度至少到玩家腰部以上（>= 1.5 单位）

### Scenario: 碰撞检测仍正常工作
- **GIVEN** 障碍物高度已调整
- **WHENContent** 玩家走向障碍物
- **THEN** 碰撞检测正确阻止玩家穿过障碍物登

### Scenario: 射击损坏检测正常
- **GIVEN** 障碍物高度已调整
- **WHEN** 玩家射击障碍物
- **THEN** 伤害检测正常，障碍物可被破坏
