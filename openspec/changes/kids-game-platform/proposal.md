## Why

为儿童打造一个安全、有趣的纯 H5 游戏平台，降低用户获取门槛（无需下载安装，浏览器即开即玩）。首期实现一款类似 CS 的第一人称射击游戏，采用 Q 版敌人和卡通化视觉风格，确保内容适合儿童群体。所有数据本地存储，无需后端服务器。

## What Changes

- 新建一个基于 Vue + Vite 的儿童游戏平台项目
- 实现 3D 第一人称射击游戏（类似 CS 风格，但卡通 Q 版化）
- 实现武器系统：多种可切换武器、倍镜配件
- 实现敌人 AI：Q 版形象、来回巡逻走动、血量实时显示
- 实现玩家自定义：可更换武器、倍镜、人物形象
- 实现平台首页：游戏列表入口（可扩展更多游戏）
- 实现本地存储：游戏进度、自定义配置、最高分等数据持久化

## Capabilities

### New Capabilities

- `game-platform-core`: 游戏平台核心功能，包括首页导航、游戏列表入口、平台框架
- `fps-game-3d`: 3D 第一人称射击游戏核心玩法，包含场景渲染、相机控制、移动系统
- `weapon-system`: 武器系统，包含多种武器切换、倍镜切换、武器属性（伤害、射速、弹夹容量）
- `enemy-ai-combat`: 敌人 AI 与战斗系统，包含 Q 版敌人模型、巡逻走动、血量系统、击中反馈
- `player-customization`: 玩家自定义系统，包含角色形象更换、武器更换、配置保存
- `local-storage`: 本地数据存储，包含游戏进度、配置信息、成就系统
- `child-ui-theme`: 儿童友好的 UI 主题，包含色彩方案、字体、动画效果

## Impact

- **新建整个项目**：Vue 3 + Vite 前端项目
- **3D 渲染引擎**：需要引入 Three.js 或 Babylon.js 作为 H5 3D 渲染方案
- **无后端依赖**：全部数据使用浏览器 localStorage / IndexedDB 存储
- **资源文件**：需要 3D 模型（Q 版敌人、武器模型）、音效、贴图资源
- **浏览器兼容性**：需要确保在主流现代浏览器中运行良好（Chrome、Edge、Safari）
