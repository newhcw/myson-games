## ADDED Requirements

### Requirement: 治疗者敌人类型

系统SHALL新增"治疗者"敌人类型，不攻击玩家，而是周期性为周围友军恢复生命值。

#### Scenario: 治疗者基础属性

- **WHEN** 游戏生成治疗者
- **THEN** 使用配置：血量=60，移速=2，视野距离=15，视野角度=360°（全向）
- **AND** 外观：绿色身体（0x00FF7F）+ 白色十字标记 + 绿色脉冲光晕
- **AND** 不执行攻击行为（`damage=0`，`attackInterval=999999`）

#### Scenario: 治疗者巡逻行为

- **WHEN** 治疗者未发现玩家
- **OR** 治疗者不处于被威胁状态
- **THEN** 执行正常巡逻行为
- **AND** 不主动追逐玩家

### Requirement: 治疗者治疗机制

治疗者SHALL每3秒检测周围8米内的友军，为血量不满的友军恢复20点生命值。

#### Scenario: 周期性治疗检测

- **WHEN** 治疗者存活
- **AND** 距离上次治疗超过3秒
- **THEN** 检测周围8米内的所有友军敌人（`type !== 'healer'`）
- **AND** 对血量 < maxHealth 的友军施加治疗

#### Scenario: 治疗执行

- **WHEN** 找到需要治疗的友军
- **THEN** 恢复该友军20点生命值（不超过maxHealth）
- **AND** 播放绿色治疗粒子特效（从治疗者飘向被治疗者）
- **AND** 被治疗者身体短暂闪绿反馈

#### Scenario: 无友军需要治疗

- **WHEN** 治疗者触发治疗检测
- **AND** 周围8米内无友军或所有友军血量满
- **THEN** 不执行治疗
- **AND** 不播放治疗特效

### Requirement: 治疗者视觉效果

治疗者SHALL具有明确的视觉标识，让玩家能快速识别并优先击杀。

#### Scenario: 治疗者常驻特效

- **WHEN** 治疗者存活
- **THEN** 身体周围有绿色脉冲光晕（持续动画）
- **AND** 头顶显示绿色十字图标（"+")

#### Scenario: 治疗释放特效

- **WHEN** 治疗者执行治疗
- **THEN** 从治疗者向被治疗者发射绿色粒子流
- **AND** 被治疗者血条短暂显示绿色"+20"数字

#### Scenario: 治疗者血条颜色

- **WHEN** 显示治疗者血条
- **THEN** 血条颜色使用绿色主题（区别于小兵蓝、精英紫、BOSS红）
- **AND** 类型标签显示"治疗者"

### Requirement: 治疗者波次配置

治疗者SHALL在第6波或更高波次加入游戏。

#### Scenario: 第6波开始出现的的治疗者

- **WHEN** 波次配置为第6波或更高
- **THEN** 波次配置可以包含 `type: 'healer'`
- **AND** 建议第6波：1个治疗者 + 5个小兵 + 1个精英
- **AND** 建议第9波：2个治疗者 + 3个精英 + 2个BOSS

#### Scenario: 治疗者击杀得分

- **WHEN** 玩家击杀治疗者
- **THEN** 获得200分（高于普通敌人，反映其战略价值）
